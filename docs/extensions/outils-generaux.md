# Outils généraux

Ces extensions nécessitent peu ou pas de configuration spécifique à
l'environnement zdev.

| Extension                         | Rôle                                                          |
|-----------------------------------|---------------------------------------------------------------|
| `timonwong.shellcheck`            | Analyse statique de scripts Bash                             |
| `mkhl.shfmt`                      | Formatage de scripts Shell                                    |
| `ms-python.python`                | Support Python (langue, débogage, tests)                     |
| `ms-python.debugpy`               | Débogueur Python                                             |
| `ms-python.vscode-pylance`        | Serveur de langage Python (Pyright)                          |
| `ms-python.vscode-python-envs`    | Gestion des environnements virtuels                          |
| `charliermarsh.ruff`              | Linter et formateur Python (ultra-rapide)                    |
| `tamasfe.even-better-toml`        | Support TOML (pyproject.toml, Cargo.toml…)                  |
| `ZainChen.json`                   | Validation et navigation JSON                                |
| `redhat.vscode-yaml`              | Validation YAML avec schémas (OpenAPI, k8s…)                |
| `redhat.vscode-xml`               | Support XML avec validation                                  |
| `mechatroner.rainbow-csv`         | Coloration des colonnes CSV/TSV                               |
| `PKief.material-icon-theme`       | Icônes de fichiers Material Design                           |
| `PKief.material-product-icons`    | Icônes d'interface VS Code                                   |
| `Catppuccin.catppuccin-vsc`       | Thème de couleurs                                            |
| `GitHub.copilot`                  | Complétion de code IA (abonnement GitHub requis)             |
| `GitHub.copilot-chat`             | Assistant IA conversationnel (abonnement GitHub requis)      |
| `mhutchie.git-graph`              | Visualisation graphique de l'historique Git                  |
| `yzhang.markdown-all-in-one`      | Aperçu, table des matières, raccourcis Markdown              |
